# 🌦️ Previsão do Tempo — Backend

Este repositório contém a base lógica do projeto de previsão do tempo, desenvolvido com Node.js e Express. Ele consome uma API pública para fornecer dados climáticos que alimentam o Frontend da aplicação.

## 🚀 Tecnologias utilizadas

- Node.js
- Express
- Axios
- Dotenv

## 📦 Funcionalidades

- Consumo de dados de clima via API pública
- Organização de rotas para entrega de dados ao Frontend
- Estrutura modular e clara para fácil manutenção

## ✅ Testes

Todas as rotas foram testadas manualmente via terminal e ferramentas como Postman, garantindo o funcionamento correto da aplicação.
